# Seminario APP

Aplicacion para el seminario de software educativo, en el cual se busca fortalecer la enseñanza y el aprendizaje del abecedario en el idioma ingles.

## Instalacion

```bash
git clone https://github.com/vprada25/ABC_APP.git
flutter create .
```
